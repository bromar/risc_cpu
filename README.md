<h1 align="center">RISC Processor</h1>
&nbsp;

A simple 32-bit RISC processor capable of handling arithmetic and logical operations.


&nbsp;
<h4><em>Instructions Supported</em></h4>
* LW(Load Word)
* SW(Store Word)
* BEQ(Branch Equal)
* ADD
* ADDU
* ADDI
* SUBI
* AND
* OR
* NOR
* NOT
* SLT(Set on Less than)
